# Snake Game (Python Turtle)

A classic Snake Game built with Python and the Turtle graphics module.
The game features smooth movement, food spawning, score tracking, and collision detection.
This project demonstrates object-oriented programming in Python, event handling, and simple game development.


<img width="901" height="716" alt="image" src="https://github.com/user-attachments/assets/b257ea4b-2978-4544-b052-9c3de8b8d532" />

## Features

- Classic Snake gameplay
- Real-time keyboard controls
- Random food generation
- Score tracking system
- Collision detection (wall & self)
- Object-Oriented Programming structure
- Smooth animation using screen.tracer()

## Technologies Used

- Python
- Turtle Graphics Module
- OOP (Object-Oriented Programming)

## Requirements

Before running the game, make sure you have: Python 3.x installed

You can download Python from: https://www.python.org/downloads/

To check if Python is installed:
For Windows
``` bash
python --version
```
To check if Python is installed:
For MacBook
``` bash
python3 --version
```


## How to Run the Game

Follow these steps to run the game on your computer.

1. Clone the Repository
 ``` bash
git clone https://github.com/adebayoadesugba/Python_Snake_Game.git
```
2. Navigate into the Project Folder
``` bash
cd snake-game
```
3. Run the Game on Windows
``` bash
python main.py
```

MacBook

``` bash
python3 main.py
```

The game window should open and you can start playing.

## Controls

| Key            | Action     |
| -------------- | ---------- |
| ⬆️ Arrow Up    | Move Up    |
| ⬇️ Arrow Down  | Move Down  |
| ⬅️ Arrow Left  | Move Left  |
| ➡️ Arrow Right | Move Right |

## Game Rules

- Eat the food to grow the snake
- Each food increases your score
- Avoid hitting the wall
- Avoid colliding with your own body
